## Overview
Welcome to my personal web portfolio! This portfolio showcases my skills, projects, and experience as a web developer. It is built with **HTML5, CSS3, SASS**, and **Bootstrap 5** for responsive design.

## Features
- **Semantic HTML5 structure** for accessibility and SEO.
- **Responsive design** compatible with desktop, tablet, and mobile devices.
- **Reusable CSS variables** for color themes and scalable styling.
- **Project detail pages** for showcasing individual work.
- **Contact form** for direct communication.
- **Consistent navigation and footer** across all pages.
- **SEO optimized** titles and meta descriptions.
- **Accessible** with descriptive alt text, ARIA attributes, and proper links.

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/Marttify/Porfolio-Dev.git
````
```bash
cd Porfolio-Dev
````
```bash
npm install
````
```bash
sass --watch assets/sass/main.scss assets/css/main.css
````

2. Open `index.html` in your preferred browser.
3. Navigate through sections: Home, About Me, Projects, Skills, and Contact.
